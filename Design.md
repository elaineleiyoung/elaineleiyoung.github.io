# Personal Website Asset Pack — Daoist / Wuxia / Literati Aesthetic

## Project Goal

Create a cohesive set of high-resolution PNG assets for a personal portfolio website.

The website should feel:

- Sophisticated
- Graceful
- Scholarly
- Calm
- Refined
- Timeless
- Literati-inspired
- Daoist and wuxia influenced
- Elegant rather than decorative

The visual language should resemble:

- Song Dynasty landscape paintings
- Classical Chinese ink wash (水墨)
- Jiangnan/Suzhou water towns
- Cloud Recesses aesthetic
- Museum exhibition graphics
- Luxury editorial design
- Contemporary East Asian minimalism

---

# Avoid

Do **not** make the assets resemble:

- Anime
- Donghua
- Comics
- Game splash art
- Restaurant menus
- Tea packaging
- Tourism advertisements
- PowerPoint templates
- Marketing graphics
- Decorative clip art
- Fantasy RPG concept art

This is a professional personal website, not a commercial product.

---

# Core Design Philosophy

Every asset should embody:

- Negative space
- Restraint
- Balance
- Softness
- Atmosphere
- Elegance

Prioritize mood over detail.

A visitor should feel:

> "This person is thoughtful, disciplined, creative, and intellectually curious."

rather than:

> "This person likes fantasy artwork."

---

# Technical Requirements

## File Format

All assets should be delivered as:

- Transparent PNGs

No paper textures.

No textured backgrounds.

No borders baked into the image unless explicitly requested.

---

## Resolution

### Small Assets

Minimum:

- 900px wide

Preferred:

- 1200–1600px wide

---

### Hero Assets

Minimum:

- 2500px wide

Preferred:

- 3000–4000px wide

---

## Quality

Designed for:

- Retina displays
- High DPI screens
- Responsive websites

Ink textures should remain crisp while preserving soft watercolor transitions.

# Composition Rules

Every asset should:

- Use extensive negative space
- Feel editorial rather than illustrative
- Support text overlays
- Avoid visual clutter
- Favor elegance over realism
- Maintain consistent visual language

Assets should feel like pieces of the same design system.

Not independent artworks.
